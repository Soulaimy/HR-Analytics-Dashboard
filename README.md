# HR Analytics Dashboard

## Contexte

Projet d'analyse RH réalisé avec Power BI.

L'objectif est d'analyser les effectifs, les salaires, les performances, les promotions et l'absentéisme afin d'aider les équipes RH à prendre des décisions basées sur les données.

---

## Technologies utilisées

- Power BI
- Excel
- Python
- Pandas
- GitHub

---

## KPI suivis

- Nombre d'employés
- Salaire moyen
- Nombre de promotions
- Nombre d'absences
- Taux d'absentéisme

---

## Analyses réalisées

### Répartition des employés par département

Identification des départements les plus représentés.

### Salaire moyen par département

Comparaison des rémunérations moyennes.

### Performance par département

Analyse des performances des équipes.

### Répartition des postes

Visualisation de la structure organisationnelle.

---

## Résultats

- Dashboard interactif Power BI
- Analyse RH complète
- Reporting visuel
